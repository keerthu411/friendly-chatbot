# Friendly Chatbot

Friendly Chatbot is a small web-based chatbot project that I created using HTML, CSS, and JavaScript. The main idea of this project is to allow users to type a message and get a simple automated reply from the bot.

## Project Objective

The objective of this project is to build a basic chatbot that can have simple conversations with users through a web interface.

## Features

* Clean and simple chat interface
* Users can type their message in the input box
* The chatbot replies automatically
* Supports basic greetings and simple questions
* Can tell a joke
* Works directly in the browser without any installation

## Technologies Used

* HTML
* CSS
* JavaScript

## Approach Used

For this project, I used a rule-based approach. The chatbot does not use AI or machine learning. Instead, it checks the message typed by the user and gives a response based on matching keywords.

For example, when the user types "hello" or "hi", the chatbot gives a greeting. When the user asks for a joke, it gives a predefined joke.

I chose this approach because it is simple, beginner-friendly, and suitable for understanding how chatbot conversations work at a basic level.

## What Makes This Chatbot Unique

This chatbot is lightweight and easy to use. It does not need an API key, database, or backend server. Since it is made only with HTML, CSS, and JavaScript, it can run directly in the browser and can also be hosted easily using GitHub Pages.

## Challenges Faced

One challenge I faced was deciding how the chatbot should understand different user messages. I solved this by using keyword checking in JavaScript.

Another challenge was designing the chat layout in a neat way. I used CSS to create separate message bubbles for the user and the chatbot, which made the conversation easier to read.

## How to Run the Project Locally

1. Clone this repository:

```bash
git clone your-repository-link
```

2. Open the project folder:

```bash
cd friendly-chatbot
```

3. Open the `index.html` file in a browser.

You can also run the project using the Live Server extension in VS Code.
